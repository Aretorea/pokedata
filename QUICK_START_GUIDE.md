# 快速启动指南 - 宝可梦自动变身功能

## 🚀 快速测试步骤

### 方法1: 使用游戏直接测试

1. **启动游戏**
   ```bash
   cd /workspace/pokemon-tower
   python3.11 -m http.server 8080
   ```

2. **打开浏览器**
   - 访问: http://localhost:8080

3. **开始游戏**
   - 点击"开始游戏"
   - 选择角色（推荐：赤红）
   - 选择初始宝可梦（推荐：皮卡丘）

4. **测试自动变身**
   - 进入第一个战斗节点
   - 观察是否自动变身为第一只宝可梦
   - 检查玩家精灵是否变成宝可梦

5. **测试排序功能**
   - 点击地图界面的"查看宝可梦"
   - 拖拽宝可梦卡牌调整顺序
   - 观察第一张卡牌是否高亮显示
   - 进入战斗测试是否使用了新的首发宝可梦

### 方法2: 使用测试页面

## 🐛 常见问题排查

### 问题1: 自动变身不生效
**原因**: 可能没有宝可梦牌
**解决**: 检查 GameState.player.pokemonCards 是否为空

### 问题2: 排序后顺序没有改变
**原因**: 可能只有一只宝可梦
**解决**: 捕捉更多宝可梦（至少2只）

### 问题3: 拖拽不工作
**原因**: 浏览器不支持拖拽
**解决**: 使用现代浏览器（Chrome/Firefox/Safari）

### 问题4: 宝可梦牌在战斗中抽不到
**原因**: 这是正常设计
**解决**: 自动变身的宝可梦会从抽牌堆移除

## 💡 使用技巧

1. **首发宝可梦选择**
   - 选择高攻击的宝可梦作为首发
   - 选择有群攻技能的宝可梦
   - 考虑属性相克关系

2. **排序策略**
   - 前排放主力宝可梦
   - 后排放辅助宝可梦
   - 考虑进化顺序

3. **战斗策略**
   - 自动变身后立即可以使用技能
   - 可以在战斗中使用其他宝可梦牌切换
   - 切换宝可梦会触发返回手牌效果

## 🎮 游戏玩法建议

1. **新手推荐流程**
