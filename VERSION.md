# 宝可梦卡牌游戏版本说明

## 当前版本：v1.1.2

---

## 📋 现版本功能总结

### 🎮 游戏核心系统

#### 战斗系统
- **回合制卡牌战斗**：杀戮尖塔风格，每回合抽牌、打出卡牌、结束回合
- **能量系统**：每回合3点能量，卡牌消耗能量打出
- **护盾机制**：防御卡获得护盾，下回合重置
- **状态效果**：灼伤、麻痹、中毒、睡眠、混乱、冰冻

#### 训练家系统
| 训练家 | 被动技能 | 初始HP | 初始金币 | 初始宝可梦 |
|--------|----------|--------|----------|------------|
| 赤红 | 宝可梦大师：宝可梦技能伤害+10% | 80 | 99 | 火系御三家+皮卡丘 |
| 青绿 | 进攻意识：攻击牌伤害+1 | 70 | 150 | 火系御三家+皮卡丘 |
| 小霞 | 水流操控：水系技能伤害+20% | 65 | 80 | 水系宝可梦 |
| 小刚 | 坚如磐石：初始护盾+5，护盾效果+25% | 100 | 80 | 岩石系宝可梦 |
| 小拉 | 龙之血脉：龙系技能伤害+30%，Boss战额外+15% | 90 | 120 | 龙系宝可梦 |
| 艾莉嘉 | 草之守护：回合结束恢复3HP | 70 | 100 | 草系宝可梦 |

#### 宝可梦系统
- **20+初始宝可梦**：包含所有御三家及常见宝可梦
- **捕捉系统**：使用精灵球捕捉野生宝可梦
- **进化系统**：战斗累积进化点数，进化后属性提升
- **超进化系统**：使用超进化石进行超进化
- **属性系统**：17种属性，各有特性

#### 地图系统
- **7层地图**：每层多个节点可选
- **节点类型**：
  - ⚔️ 战斗：普通战斗
  - 💀 精英：精英战斗，掉落遗物
  - 👑 Boss：Boss战斗，掉落稀有遗物
  - 🏪 商店：购买卡牌、移除卡牌
  - ❤️ 休息：恢复HP或强化卡牌
  - ❓ 事件：随机事件
  - 📦 宝箱：获得遗物或金币

#### 属性克制系统
完整实现17种属性的克制关系：

| 攻击属性 | 克制(2倍) | 被抵抗(0.5倍) | 免疫(0倍) |

| 火 | 草、虫、冰、钢 | 水、火、岩、龙 | - |
| 水 | 火、地、岩 | 水、草、龙 | - |
| 草 | 水、地、岩 | 火、草、毒、飞、虫、龙、钢 | - |
| 电 | 水、飞 | 电、草、龙 | 地面 |
| 冰 | 草、地、飞、龙 | 火、水、冰、钢 | - |
| 格斗 | 普、冰、岩、恶、钢 | 飞、毒、虫、超、妖 | 幽灵 |
| 幽灵 | 超、幽灵 | 恶 | 普通 |
| 龙 | 龙 | 钢、妖 | 妖精 |
| ... | ... | ... | ... |

### 🃏 卡牌系统

#### 卡牌类型
| 类型 | 数量 | 说明 |
|------|------|------|
| 攻击卡 | 30+ | 造成伤害，可能有附加效果 |
| 防御卡 | 15+ | 获得护盾，抵挡伤害 |
| 技能卡 | 20+ | 特殊效果（抽牌、回血、增益等） |
| 道具卡 | 10+ | 可消耗，使用后移除 |

#### 卡牌稀有度
| 稀有度 | 颜色 | 获取难度 | 说明 |
|--------|------|----------|------|
| 普通 | 灰色 | 容易 | 基础卡牌 |
| 罕见 | 绿色 | 普通 | 略强于普通 |
| 稀有 | 蓝色 | 较难 | 强力卡牌 |
| 史诗 | 紫色 | 困难 | 非常强力 |
| 传说 | 金色 | 极难 | 最强卡牌 |

#### 特殊卡牌
- **宝可梦卡牌**：使用后变身为对应宝可梦，获得属性加成
- **道具卡牌**：使用后从牌组永久移除（消耗品）
- **超进化卡牌**：需要特定条件才能使用

### 🏆 遗物系统

#### 遗物获取
- 精英战斗胜利：从2个稀有遗物中选择
- Boss战斗胜利：从3个传说遗物中选择
- 特殊事件：可能获得独特遗物

#### 遗物类型
- **战斗遗物**：战斗开始时生效
- **回合遗物**：每回合触发效果
- **触发遗物**：满足条件时触发
- **被动遗物**：永久生效

### 🎨 UI系统

#### 界面设计
- **竖版紧凑卡牌**：110px宽，150px高
- **动态星空背景**：闪烁星星动画
- **覆盖层界面**：奖励、商店、游戏结束等界面居中显示
- **状态效果**：显示在精灵头顶
- **血条颜色**：健康(绿)→中等(黄)→危险(红+闪烁)

#### 图片显示
- 训练家使用emoji图标
- 宝可梦使用PokeAPI官方图片
- 图片加载失败自动回退到emoji

## 🔧 开发说明

### 技术栈
- **前端**：纯HTML + CSS + JavaScript
- **样式**：原生CSS3
- **无框架**：不依赖任何前端框架
- **数据存储**：LocalStorage

### 关键文件说明

#### battle.js - 战斗系统核心
```javascript
// 主要函数
startBattle(enemyData)      // 开始战斗
playCard(index)             // 打出卡牌
endTurn()                   // 结束回合
dealDamageToEnemy(damage, attackType, options)  // 对敌人造成伤害
dealDamageToPlayer(damage)  // 对玩家造成伤害
checkBattleEnd()            // 检查战斗结束
endBattle(victory)          // 结束战斗

// 属性克制
TYPE_CHART                  // 属性克制表
getTypeEffectiveness(attackType, defenderTypes)  // 计算属性克制倍率
```

#### map.js - 地图系统核心
```javascript
// 主要函数
generateMap()               // 生成地图
renderMap()                 // 渲染地图
selectMapNode(nodeId)       // 选择节点
executeNodeEvent(node)      // 执行节点事件
showRewardScreen(isElite)   // 显示奖励界面
```

#### ui.js - 界面控制核心
```javascript
// 主要函数
showScreen(screenId)        // 显示界面
updateUI()                  // 更新UI
updateBattleUI()            // 更新战斗UI
renderHand()                // 渲染手牌
createCardElement(card, index)  // 创建卡牌元素
```

### 添加新宝可梦
在 `data/pokemon-data.js` 中添加：
```javascript
{
    id: 'pokemon-id',
    name: '宝可梦名称',
    types: ['fire'],           // 属性
    hp: 50,                    // 基础HP
    attack: 60,                // 物攻
    spAttack: 50,              // 特攻
    defense: 40,               // 物防
    spDefense: 40,             // 特防
    speed: 65,                 // 速度
    sprite: '🔥',              // emoji图标
    ability: 'blaze',          // 特性ID
    abilityName: '猛火',
    skills: [                  // 技能列表
        { name: '火花', damage: 6, attackType: 'fire', cost: 1 }
    ]
}
```

### 添加新卡牌
在 `data/cards-data.js` 中添加：
```javascript
{
    id: 'card-id',
    name: '卡牌名称',
    type: 'attack',            // attack/defense/skill/item
    cost: 1,                   // 能量消耗
    rarity: 'common',          // 稀有度
    pokemonType: 'fire',       // 属性（可选）
    description: '描述文本',
    effects: [
        { type: 'damage', value: 6, attackType: 'fire' },
        { type: 'applyStatus', status: 'burn', chance: 0.5 }
    ]
}
```

### 添加新遗物
在 `data/relics-data.js` 中添加：
```javascript
{
    id: 'relic-id',
    name: '遗物名称',
    description: '描述文本',
    rarity: 'common',          // common/rare/boss
    sprite: '💎',              // emoji图标
    effect: {
        type: 'damage',        // 效果类型
        value: 3               // 效果数值
    }
}
```
